# 📄 AI PDF Chatbot (LangChain + RAG)

This is a simple AI chatbot that allows users to upload PDF files and ask questions from them using AI.

## 🚀 Features
- PDF upload
- AI Q&A from documents
- RAG (Retrieval Augmented Generation)
- Vector database search
- Streamlit UI

## 🧠 Tech Stack
- LangChain
- OpenAI GPT-4o-mini
- Chroma Vector DB
- Streamlit
- Python
